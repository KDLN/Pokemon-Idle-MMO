### Box Panel (Pokemon Storage)
- Grid layout of stored Pokemon
- Selection checkboxes
- Sorting controls (by level, name, type)
- Filter by type
- Transfer to party action
- Release Pokemon (with confirmation)

### Pokedex Panel
- Species list with catch status
- Silhouette for uncaught
- Full art for caught
- Type filter tabs
- Catch count and percentage

### Boost/Buff Cards
- Active buff icon and name
- Countdown timer (MM:SS / HH:MM:SS)
- Red + pulse animation when <1 min
- Buff effect description
- "Expired" state

### Item Card
- Item icon/image
- Item name and description
- Quantity owned
- Use/Equip button
- Sell option

### Currency Display
- Gold/coins amount
- Premium currency amount
- Animated increment on gain
