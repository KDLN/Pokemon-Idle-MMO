## v1 Requirements

### Guild Foundation

- [x] **GUILD-01**: Player can create a guild with name (unique), tag (3-5 chars), and description
- [x] **GUILD-02**: Guild enforces 50 member maximum cap
- [x] **GUILD-03**: Player can view list of guilds with search/filter
- [x] **GUILD-04**: Player can join an open guild directly
- [x] **GUILD-05**: Player can leave a guild (with 24hr cooldown before joining another)
- [x] **GUILD-06**: Player can view guild member roster with online status, role, and last active
- [x] **GUILD-07**: Guild founder automatically becomes Leader role

### Guild Roles & Permissions

- [x] **ROLE-01**: Guild has three roles: Leader, Officer, Member
- [x] **ROLE-02**: Leader can promote Members to Officer
- [x] **ROLE-03**: Leader can demote Officers to Member
- [x] **ROLE-04**: Leader can transfer leadership to another member
- [x] **ROLE-05**: Leader can kick any member (Officer or Member)
- [x] **ROLE-06**: Officer can kick Members (not other Officers)
- [x] **ROLE-07**: Leader can disband guild (requires confirmation)

### Guild Invites

- [x] **INVITE-01**: Leader/Officer can send invite to any player not in a guild
- [x] **INVITE-02**: Player receives invite notification
- [x] **INVITE-03**: Player can accept or decline invite
- [x] **INVITE-04**: Invites expire after 7 days
- [x] **INVITE-05**: Guild can set join mode: Open, Invite-Only, or Closed

### Guild Chat

- [x] **CHAT-01**: Guild has dedicated chat channel visible only to members
- [x] **CHAT-02**: Guild chat messages show player name, role badge, and timestamp
- [x] **CHAT-03**: Guild chat persists history (last 100 messages)

### Guild Bank - Currency

- [x] **BANK-01**: Guild has shared currency pool (separate from member wallets)
- [x] **BANK-02**: Any member can deposit currency
- [x] **BANK-03**: Officer can withdraw currency up to daily limit
- [x] **BANK-04**: Leader can withdraw currency without limit
- [x] **BANK-05**: Members cannot withdraw currency

### Guild Bank - Items

- [x] **BANK-06**: Guild bank has item storage (slots for potions, balls, etc.)
- [x] **BANK-07**: Any member can deposit items
- [x] **BANK-08**: Officer can withdraw items up to daily limit
- [x] **BANK-09**: Leader can withdraw items without limit
- [x] **BANK-10**: Members cannot withdraw items

### Guild Bank - Pokemon

- [x] **BANK-11**: Guild bank has Pokemon storage (limited slots)
- [x] **BANK-12**: Any member can deposit Pokemon (all Pokemon allowed for now)
- [x] **BANK-13**: Officer can withdraw Pokemon up to daily limit
- [x] **BANK-14**: Leader can withdraw Pokemon without limit
- [x] **BANK-15**: Members cannot withdraw Pokemon

### Guild Bank - Logs

- [x] **LOG-01**: All bank transactions are logged (who, what, when, action)
- [x] **LOG-02**: Leader and Officers can view full transaction log
- [x] **LOG-03**: Members can view their own transactions only

### Guild Quests

- [x] **QUEST-01**: Guild has daily quests (reset at midnight UTC)
- [x] **QUEST-02**: Guild has weekly quests (reset Monday midnight UTC)
- [x] **QUEST-03**: Quest types: catch X Pokemon, catch X of type, battle X times, evolve X Pokemon
- [x] **QUEST-04**: All member activity counts toward guild quest progress
- [x] **QUEST-05**: Quest progress is visible to all members
- [x] **QUEST-06**: Completed quests reward guild bank (currency, items, or guild points)

### Guild Shop

- [x] **SHOP-01**: Guild can spend bank currency/points on buffs
- [x] **SHOP-02**: Buff: +10% XP for all members (1 hour)
- [x] **SHOP-03**: Buff: +10% catch rate for all members (1 hour)
- [x] **SHOP-04**: Buff: +10% encounter rate for all members (1 hour)
- [x] **SHOP-05**: Active buffs are visible to all members
- [x] **SHOP-06**: Only Leader/Officer can purchase buffs

### Guild Statistics & Leaderboard

- [x] **STATS-01**: Guild displays total Pokemon caught (all members, all time)
- [x] **STATS-02**: Guild displays total unique species caught (guild Pokedex)
- [x] **STATS-03**: Guild displays member count and average level
- [x] **STATS-04**: Guild displays age (days since creation)
- [x] **STATS-05**: Guild leaderboard ranks guilds by configurable metric (catches, Pokedex, members)
- [x] **STATS-06**: Player can view top 50 guilds on leaderboard

### Zone Content

- [x] **ZONE-01**: Cerulean City zone (town, connected to Route 4 and Routes 24/25)
- [x] **ZONE-02**: Misty's Gym in Cerulean City (Water-type, Cascade Badge)
- [x] **ZONE-03**: Route 24 zone (Nugget Bridge, north of Cerulean)
- [x] **ZONE-04**: Route 25 zone (Bill's House area, east of Route 24)
- [x] **ZONE-05**: Route 24-25 Pokemon: Bellsprout, Oddish, Abra, Slowpoke, Venonat, Pidgey
